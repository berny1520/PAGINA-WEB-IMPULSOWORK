IMPULSOWORK Training OTEC - Sitio web listo para publicar

Contenido:
- index.html
- carpeta assets/ con logo y fotos reales
- brochure PDF
- certificado SCI PDF

Cómo verlo:
1. Descomprime el ZIP.
2. Abre index.html en tu navegador.

Cómo publicarlo online:
Opción 1: GitHub Pages
1. Crea una cuenta en GitHub.
2. Crea un repositorio nuevo.
3. Sube todo el contenido de esta carpeta.
4. En Settings > Pages, activa publicación desde la rama main y carpeta root.
5. GitHub te entregará una URL pública.

Opción 2: Netlify
1. Entra a netlify.com
2. Arrastra la carpeta descomprimida al panel de deploy.
3. Obtendrás una URL pública inmediata.

Opción 3: Hosting tradicional
1. Sube todos los archivos por cPanel o FTP.
2. Deja index.html en la carpeta public_html.
